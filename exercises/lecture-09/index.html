Examples of the CSS float Property Below are some examples of how the CSS float property works.

Абсолютне позиціонування виносить елемент із звичайного потоку документа, фіксуючи його положення відносно батьківського елемента або документа.

Статичне позиціонування є типовим для кожного елемента. Це просто означає, помістіть елемент у звичайне положення в документі.

Відносне позиціювання. Положення елемента встановлюється відносно його поточного положення. Додавання властивостей left, top, right і bottom змінює позицію елемента і зрушує його в ту чи іншу сторону. Ця властивість дозволяє змінювати позицію елемента без зміни макету.
